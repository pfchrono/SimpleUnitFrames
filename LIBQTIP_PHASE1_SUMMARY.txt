## Known Limitations & Workarounds

### Limitation 1: Snapshot Data
- **Issue:** Tooltip shows frame state when created (not real-time updates)
- **Why:** Real-time updates would require polling (CPU overhead)
- **Workaround:** Hover again to refresh tooltip data

### Limitation 2: Frame Visibility Dependent
- **Issue:** Hidden frames show "—" for health/power
- **Why:** Values only update when frame visible
- **Workaround:** Expected behavior; not a bug

### Limitation 3: Instance Restrictions
- **Issue:** Some APIs may be restricted in dungeons/raids
- **Why:** WoW 12.0.0+ security model
- **Workaround:** Shows "—" for restricted data (graceful fallback)
